다시 기술 블로그를 시작해봅니다 :D

  1. 단계별로 hexo 설치 npm install -g hexo-cli
  2. hexo blog repo 만들기 hexo init <blog_repo_name>
  3. 테마 설치 블로그 루트폴더 내에서 git clone https://github.com/ppoffice/hexo-theme-hueman.git themes/hueman 실행합니다
  4. 테마 설치후 테마 파일에 _config.yml.example 파일을 .example을 없애주어서 활성하시켜줍니다
  5. 루트 폴더에 _config.yml파일로 가서 deploy할 세팅을 해줍니다

…맙소사 amorfati0310/amorfati0310.io 이 repo가 아니고 다른 repo에서는 계속 테마가 적용이 안되고 링크도 이상하게 적용되는 상황이 발생했는데 이걸로 시간 얼마나 잡아먹었는지 ㅜㅜ
왜 다시 잘 되는지는 모르겠다…. 아마도 master로 올라가는데 적용되는 시간이 좀 있었지 않았나 싶다.;ㅁ;
